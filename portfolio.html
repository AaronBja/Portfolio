TechStore - E-Commerce

React Node.js MongoDB Stripe

📋 Description

Plateforme de réservation immobilière (clone Airbnb) moderne et évolutive. Elle repose sur une architecture Full Stack robuste combinant Spring Boot 3 pour le backend et d'Angular 17 pour une expérience utilisateur fluide.

✨ Fonctionnalités

  • Création complète de propriétés par les hôtes (photos, descriptions, tarifs).
  • Filtres multicritères par destination, dates de séjour et nombre de voyageurs.
  • Visualisation des logements sur une carte dynamique via l'API Leaflet.
  • Tunnel de réservation complet avec gestion des disponibilités en temps réel.
  • Intégration d'Auth0 pour une gestion robuste des utilisateurs et des rôles (Hôte/Voyageur).
  • Dashboards dédiés pour le suivi des réservations et la gestion des biens en location.
  • Design moderne et adaptatif optimisé pour tous les écrans.

🛠️ Technologies

  • Frontend: Angular 17 (Standalone Components, Signals), PrimeNG, PrimeFlex, RxJS.
  • Backend: Java 21, Spring Boot 3, Spring Security, Spring Data JPA.
  • Base de données: PostgreSQL, Liquibase (versioning de base de données).
  • Paiement: Auth0 (OAuth 2.0 / OIDC), JWT.
  • Styling: Leaflet API, MapStruct, Maven, FontAwesome.
